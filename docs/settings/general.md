# 일반 설정

## 환경설정 열기

**환경설정** 메뉴에서 다양한 설정을 변경할 수 있습니다.

## 암호화 키

계정 정보 파일(*.dat) 암호화에 사용되는 키를 변경합니다.

!!! warning "주의"
    암호화 키를 변경하면 이전 키로 암호화된 파일을 읽을 수 없습니다.

## 게임 정보

게임 생성 시 사용되는 정보를 설정합니다.

| 항목 | 설명 | 예시 |
|------|------|------|
| 게임 제목 (Name) | 게임 이름 접두어 | MAG |
| 비밀번호 (Password) | 게임 비밀번호 | 0402 |
| 게임 번호 (Count) | 현재 게임 번호 | 1 |

### 게임 이름 생성 규칙

`게임 제목 + 게임 번호` 형식

예: 게임 제목이 `MAG`, 번호가 `11`이면 → `MAG11`

번호 자릿수는 설정파일의 `digit_size`로 조정:

```ini
digit_size = 2  # MAG01 ~ MAG99
digit_size = 3  # MAG001 ~ MAG999
```

## 스케줄

주기적으로 실행할 스크립트를 설정합니다.

| 항목 | 설명 |
|------|------|
| 사용 여부 | 스케줄 활성화 |
| 스케줄 명 | 식별용 이름 |
| 실행 간격 | 초 단위 |
| 스크립트 | 실행할 스크립트명 |

**타이머 사용** 메뉴로 스케줄을 실행합니다.

!!! tip "Caps Lock 일시정지"
    Caps Lock이 켜져 있으면 스케줄이 실행되지 않습니다.
    마을에 들릴 때 활용하세요.

## 패널 표시

### 사이드 패널

계정 입력/수정/삭제 패널을 표시하거나 숨깁니다.

### 하단 패널

실행 옵션 및 실행 버튼 패널을 표시하거나 숨깁니다.

## 항상 위에

프로그램 창을 항상 최상단에 표시합니다.

설정 방법:

- 메뉴: **환경설정 → 항상 위에**
- 설정파일: `always_on_top = True`
- 레지스트리: `HKEY_CURRENT_USER\SOFTWARE\Codebar\d2r_gui` → `always on top`

## 단축키 활성화

게임 생성/참가 등 단축키를 활성화합니다.

!!! note "동작 조건"
    D2R 클라이언트나 이 프로그램 창이 활성화된 상태에서만 동작합니다.

## 개인정보 마스킹

이메일 정보를 마스킹 처리합니다.

마스킹 비율 조정:

```ini
masking_rate = 0.2  # 20%만 표시
```

## 다크 모드

어두운 테마로 변경합니다.

!!! note "재시작 필요"
    윈도우 타이틀바는 OS에서 관리하므로 프로그램 재시작 후 완전히 적용됩니다.

## 언어 설정 (Localization)

윈도우 언어 설정에 따라 자동 변경됩니다.

현재 지원 언어:

- 한국어
- 영어

## 소리 활성화

다음 상황에서 소리를 재생합니다:

- 프로그램 시작
- 오류 발생
- 소개 페이지
- 스크립트 단계
- 종료

## 환경설정 저장/읽기

| 메뉴 | 설명 |
|------|------|
| 현재 설정을 파일로 저장 | 설정을 d2r.ini로 저장 |
| 파일에서 설정 읽기 | d2r.ini에서 설정 로드 |
