# 창 관리

## 창 제목 설정

D2R 클라이언트 창의 제목을 사용자가 지정한 설명으로 변경할 수 있습니다.

### 설정 방법

1. 하단 패널에서 **제목설정** 체크
2. 계정 등록 시 **설명** 항목 입력
3. 클라이언트 실행

### 표시 형식

- **설명 입력 시**: 입력한 설명 표시
- **설명 미입력 시**: `사용자명, 모드, 서버, 설정파일` 형식

### 개인정보 마스킹

이메일 정보를 마스킹 처리할 수 있습니다.

**환경설정 → 개인정보 마스킹**

마스킹 비율 조정: 설정파일(d2r.ini)의 `masking_rate = 0.2`

- 0.2 = 20%만 표시
- 0.5 = 50%만 표시

## 창 정렬

여러 클라이언트 창을 자동으로 정렬합니다.

### 자동 정렬

하단 패널의 **창 정렬** 체크 후 실행하면:

1. 저장된 레이아웃이 있으면 해당 레이아웃 적용
2. 없으면 3x2 자동 정렬

### 수동 정렬

**게임 옵션 → 창 정렬** 메뉴에서 선택한 계정들의 창을 정렬합니다.

## 레이아웃 관리

### 레이아웃 관리자 열기

**도구 → 레이아웃 관리자**

![레이아웃 관리자](../images/layout-manager.png)

### 기능

| 버튼 | 설명 |
|------|------|
| 저장 | 현재 창 위치를 레이아웃으로 저장 |
| 적용 | 저장된 레이아웃 적용 |
| 삭제 | 레이아웃 삭제 |

### 레이아웃 개수

설정파일(d2r.ini)의 `layout_count = 7`로 조정

## 항상 위에

이 프로그램 창을 항상 최상단에 표시합니다.

### 설정 방법

1. **환경설정 → 항상 위에** 체크
2. 또는 설정파일: `always_on_top = True`

## 클라이언트별 설정파일

각 클라이언트마다 다른 비디오/오디오 설정을 사용할 수 있습니다.

### 설정파일 위치

```
%userprofile%\Saved Games\Diablo II Resurrected\
├── Settings.json      # 기본 설정
├── Settings_1.json    # 클라이언트 1용
├── Settings_2.json    # 클라이언트 2용
└── ...
```

### 적용 방법

1. `Settings.json` 복사
2. 원하는 이름으로 저장 (예: `Settings_Low.json`)
3. 파일 내용 수정 (해상도, 음량 등)
4. 계정 설정에서 **설정파일** 선택

!!! note "동작 원리"
    실행 시 지정된 설정파일을 `Settings.json`으로 복사하고,
    종료 후 원래 파일로 복구합니다.

### 설정파일 편집

계정 설정의 설정파일 옆 **편집 아이콘**을 클릭하면 JSON 편집기가 열립니다.

## 창 크기 조정

스크립트 실행 시 클릭 좌표를 일정하게 유지하기 위해 창 크기를 조정합니다.

### 설정

설정파일(d2r.ini): `resize_window = True`

- `True`: 스크립트 실행 시 1280x720으로 창 크기 조정
- `False`: 창 크기 유지
