# 테러존 트래커

현재 및 다음 테러존 지역을 실시간으로 표시합니다.

![테러존 트래커](../images/terror-zone.png)

## 활성화

**환경설정 → 테러존 추적** 체크

활성화하면 사이드 패널 이미지 위에 테러존 정보가 표시됩니다.

## 표시 정보

| 항목 | 설명 |
|------|------|
| 현재 테러존 | 현재 활성화된 테러존 지역 |
| 다음 테러존 | 다음에 활성화될 테러존 지역 |
| 남은 시간 | 다음 테러존까지 남은 시간 |

## 데이터 출처

이 기능은 [d2emu.com](http://d2emu.com)에서 제공하는 Terror Zone Tracker API를 사용합니다.

!!! note "API 인증"
    2024년 8월 29일 이후 토큰 인증 방식으로 변경되었습니다.
    **v3.2 버전 이상**에서만 정상 동작합니다.

## 음성 안내

테러존 변경 시 음성으로 알려줍니다.

- 구글번역 TTS 사용
- 클로바더빙(CLOVA Dubbing) AI 보이스 사용

### 소리 설정

**환경설정 → 소리 활성화**에서 켜고 끌 수 있습니다.
