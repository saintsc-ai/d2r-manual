# 레이아웃 관리

여러 클라이언트 창의 위치를 저장하고 불러올 수 있습니다.

## 레이아웃 관리자 열기

**도구 → 레이아웃 관리자**

![레이아웃 관리자](../images/layout-manager.png)

## 기능

### 레이아웃 저장

1. 각 클라이언트 창을 원하는 위치로 이동
2. 레이아웃 관리자에서 레이아웃 번호 선택
3. **저장** 버튼 클릭

### 레이아웃 적용

1. 적용할 레이아웃 번호 선택
2. **적용** 버튼 클릭
3. 모든 클라이언트 창이 저장된 위치로 이동

### 레이아웃 삭제

1. 삭제할 레이아웃 번호 선택
2. **삭제** 버튼 클릭

## 레이아웃 개수 설정

기본적으로 7개의 레이아웃 슬롯이 제공됩니다.

설정파일(d2r.ini)에서 변경 가능:

```ini
layout_count = 7
```

## 자동 정렬

레이아웃이 저장되지 않은 상태에서 **창 정렬** 옵션을 사용하면:

- 3x2 그리드로 자동 정렬
- 화면 크기에 맞게 배치

## 활용 예시

### 모니터별 레이아웃

- 레이아웃 1: 메인 모니터용 (4개 클라이언트)
- 레이아웃 2: 듀얼 모니터용 (6개 클라이언트)
- 레이아웃 3: 울트라와이드용

### 용도별 레이아웃

- 레이아웃 1: 카오스 런 (4인 파티)
- 레이아웃 2: 우버 트리스트람 (8인 파티)
- 레이아웃 3: 테러존 사냥

## 팁

!!! tip "효율적인 레이아웃"
    - 메인 캐릭터 창은 크게, 부 캐릭터는 작게
    - 자주 조작하는 캐릭터는 접근하기 쉬운 위치에
    - 클라이언트별 설정파일로 해상도도 다르게 설정 가능
