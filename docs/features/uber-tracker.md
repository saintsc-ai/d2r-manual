# 우버 트래커

우버 디아블로(DClone) 상태를 지역/모드별로 추적합니다.

![우버 트래커](../images/uber-tracker.png)

## 실행 방법

**도구 → 우버 트래커**

## 표시 정보

다음 조합별로 우버 디아블로 상태를 표시합니다:

- **지역**: Americas, Europe, Asia
- **모드**: 소프트코어(SC), 하드코어(HC)
- **타입**: 래더(Ladder), 비래더(Non-Ladder)

## 상태 단계

| 단계 | 설명 |
|------|------|
| 1 | 테러가 시작되었습니다 |
| 2 | 테러가 확산되고 있습니다 |
| 3 | 테러가 약간 강해졌습니다 |
| 4 | 테러가 강해졌습니다 |
| 5 | 테러가 매우 강해졌습니다 |
| 6 | **디아블로가 출현했습니다!** |

!!! success "우버 출현"
    6단계가 되면 우버 디아블로가 출현한 것입니다.
    지옥 난이도에서 SOJ를 팔면 DClone을 만날 수 있습니다.

## 데이터 출처

[d2emu.com](http://d2emu.com)에서 제공하는 API를 사용합니다.

!!! note "API 인증"
    2024년 8월 29일 이후 API 접근에 인증이 필요합니다.
    **v3.2 버전 이상**에서 지원합니다.

## 새로고침

데이터는 자동으로 갱신되지만, 수동으로 새로고침하려면 **새로고침** 버튼을 클릭하세요.
